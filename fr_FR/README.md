# fr_FR service

## env vars

```
export LANG="C.UTF-8"
export LC_ALL="C.UTF-8"
export FLASK_APP=app.py
export FLASK_DEBUG=0
export PORT=3000 
```

## install requirements

```bash
$ pip install -r requirements.txt
```

## run test
```bash
# Add -v flag for verbose mode
pytest
```

## run service
```
$ python app.py
 ```
